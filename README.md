# Alura-Pong.js
This is my first programming project, therefore it is not 100% polished and there may be a few bugfixes and 
AI balancing adjustments that need to be made. Please issue if you find any bug or solution.

CONTROLS
W --> Left racket UP
S --> Left racket DOWN
Up Arrow --> Right racket UP
Down Arrow --> Right racket DOWN
M --> Enable Multiplayer
N --> Disable Multiplayer
